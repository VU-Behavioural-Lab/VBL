
# Ethical approval

All studies at VBL require an ethical approval. When you are filling in your study information on SONA, you will need to provide the Ethical Review Board (ERB) approval code for your study. 

{% if faculty == 'SBE'%}
You can start the ethical approval procedure by filling in the [online ethics self-check](https://vueconomics.eu.qualtrics.com/jfe/form/SV_1SKjMzceWRZIk9D){:target="_blank" rel="noopener noreferrer"}. 
If your research does not require a Ethics Review Board approval, you will automatically receive an ethical review code (e.g., `SBE3/29/2021djg590`).
If the online ethics self-check requests that your research needs additional approval, you will be asked to download a copy of your ethics application so that you can send it to the Ethics Review Board for further evaluation. For more information regarding this, see <u>**[this page](https://vu.nl/en/about-vu/faculties/school-of-business-and-economics/more-about/sbe-research-ethics-and-integrity){:target="_blank" rel="noopener noreferrer"}**</u>.
